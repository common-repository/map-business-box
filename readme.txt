Display multiple google business locations with a shortcode [mbb_shortcode]. 

== Description ==

This plugin allows users to display multiple locations via the use of a single shortcode. Users will be able to select the location, add logo, add contact details, and choose the color directly inside of their admin dashboard. The front-end display will be a location accordion that is both user friendly and great for Local SEO.


== Installation ==


1. Upload the plugin folder to the `/wp-content/plugins` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->MBB Plugin screen to configure the plugin

== Frequently Asked Questions ==

= Does this plugin have support? =

Yes! If you need support for this plugin feel free to go to our website at http://localmap.co and use the contact form. A support representative will respond promptly.

= Can I Display my business name in the map? =

Yes, you are able to use your places ID to display your location marker for your business.

= Is this plugin helpful for SEO? =

Yes, google will identify “location signals” when it crawls your website, this plugin allows you to display your location signals in an intuitive format.

== Screenshots ==

1. The admin panel allows users to select the number of location in the accordion as well as select the color theme.
2. The second step is setting the location details for your locations. You can add details such as name, number, address, and logo.
3. Simply go to your location on Google, and grab the embed code.
4. The result is an intuitive location accordion that will provide excellent location signals from your website.
